## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/cryptoprice-bot.git
   cd cryptoprice-bot
   ```
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Set up your API keys in `.config` file.
4. Run the bot:
   ```bash
   python bot.py
   ```
